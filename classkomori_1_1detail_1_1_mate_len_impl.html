Detailed Description

template<typename T>
class komori::detail::MateLenImpl< T >

MateLen と MateLen16 の実装本体。中身はほぼ同じなので一箇所にまとめる。

Template Parameters
T16ビット以上の符号なし整数型
Note
初期値として「0手よりも小さい手数」を表現したいので、実際の手数に 1 を加えた値を保持する。
