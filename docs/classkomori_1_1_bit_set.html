Detailed Description

template<typename T>
class komori::BitSet< T >

整数 T を用いたフラグ集合。

Template Parameters
T内部で用いる整数型（unsigned）

std::uint64_t のような固定幅整数を用いた2値フラグの集合。 T が N ビット変数の場合、 N 個の true/false フラグを格納できる。 状態の変更は Set()/Reset() 、状態の取得は Test() または operator[] を用いる。

使える添字は 0-base。例えば、T=uint64_t なら [0, 64) の範囲。ただし、範囲外はすべてfalse として扱う。すなわち、例えば T=uint64_t の場合、 Set(334) は何もせず、 Test(334) は常に false を返す。

Note
trivially default constructible にするために、デフォルトコンストラクタは0初期化を行わない。空の BitSet が ほしい場合、BitSet::None() を用いて初期化すること。
