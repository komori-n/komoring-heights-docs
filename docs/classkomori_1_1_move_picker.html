Detailed Description

詰将棋探索用の指し手生成器

詰将棋探索に特化した指し手生成。generateMoves() では非合法手が混じっているが、詰将棋探索では合法手以外が 混じると面倒なので、生成時点で取り除く。

Note
サイズがそこそこ大きいので、再帰探索で使用する場合はスタックオーバーフローに注意。
