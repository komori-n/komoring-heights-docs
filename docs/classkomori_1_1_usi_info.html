Detailed Description

USIプロトコルに従い探索情報（info）を整形するクラス

USIプロトコルでは、エンジンの探索情報を出力するコマンドとしてinfoを用いる。infoには出力フォーマットに関して 細かいルールが多くある。このクラスでは、そのような出力フォーマットの整形および関数間での受け渡しをするための 機能を提供する。
